## 缺點與改善建議

### 一、設計與視覺

#### 1. Hero 區塊偏弱
- 現在的 Hero 是靜態影片（自動播放）+ 標題文字，沒有視覺衝擊感
- 影片播放時底部有預設控制列，視覺雜亂（`controls` 屬性，建議拿掉）
- 沒有 `poster` 圖片，影片載入前畫面空白
- **建議**：拿掉 controls、加 poster 圖、或改用全版背景圖片搭配動態效果

#### 2. 「我們是誰」區塊缺乏層次
- 只有一張大合照 + 兩段純文字，視覺上平淡
- 沒有任何 highlight（數字、icon、引言框）讓人快速掃描重點
- **建議**：加入 3–4 個特色亮點（例如「成立 X 年」、「已培訓 X+ 學員」、「每週固定開課」），用 icon + 數字排列

#### 3. 師資頁過於空曠
- 目前只有 Sean 一張卡片，頁面看起來未完成
- 卡片寬度在桌面版撐到 `max-w-[350px]` 並靠左，右側大片空白
- **建議**：補齊其他老師，或若暫時只有 Sean，調整版型讓單卡置中，視覺更完整

#### 4. 地點頁面簡陋
- 只有文字 + Callout + Google Map iframe，沒有視覺引導
- 沒有「如何到達」的步驟圖示或交通資訊
- 地圖嵌入高度固定，在某些裝置上偏小
- **建議**：加入「交通方式」（捷運、公車、停車場）小區塊；或至少加一張外觀照讓新學員辨認

#### 5. 頁尾（Footer）非常空
- 只有 Logo + 一行版權聲明
- 缺少 LINE QR code / 聯絡方式 / 快速導覽連結
- **建議**：Footer 加入 LINE 聯絡、地址、快速連結（課表、師資、費用）

---

### 二、內容

#### 6. 課程說明不夠清楚
- 「舞蹈風格介紹」頁只有簡短說明 + YouTube，沒有說明各課程的難度定義
- 新學員看完不知道自己該從哪個 level 開始
- **建議**：加入「我適合哪個課程？」小段落，用幾個問題引導（有無舞蹈經驗、節奏感強弱等）

#### 7. Testimonials 只有文字
- 三則評語全部是長文，沒有視覺區隔（星等、學習時間、舞種標記等）
- 無法快速掃描；行動裝置上 carousel 的分頁點（dot）很小，不易操作
- **建議**：加入舞種標籤（如「Bachata 學員」）、學習時長，讓評語更具體可信

#### 8. 常見問題（FAQ）內容偏少
- 只有 4 題，且問題偏基礎（如何報名、費用、地點）
- 新學員常見疑問（需要舞伴嗎、穿什麼來上課、多久能學會）都沒有
- **建議**：擴充到 8–10 題，加入「需要舞伴嗎？」（這是 Bachata/Salsa 最常見疑問）

#### 9. 沒有「下一期開課」的明確資訊
- 課表頁有日期但沒有「報名截止日」或「下一期何時開始」的說明
- 新學員不知道現在報名能不能趕上
- **建議**：在課表頁加一行「下一期 X/X 開課，現在報名即可參加」

#### 10. 缺少活動照片 / 成果展示
- 沒有學員共舞、party 或 showcase 的照片
- 「我們是誰」那張合照是目前唯一的現場照
- 對從未接觸拉丁舞的新學員來說，無法想像上課氛圍
- **建議**：加入一個 Photo Gallery 或在各頁面穿插 2–3 張現場照

### 三、使用者體驗（UX）

#### 11. 首頁沒有直達定價的路徑
- Hero 有「立即報名」和「課程資訊」，但「課程資訊」進去是課表，不是費用
- 新學員通常第一個問的是「多少錢」，目前需要三步才能看到（首頁→課程資訊→切換到費用 tab）
- **建議**：Hero 的 CTA 按鈕改為「了解費用」或在 FAQ 更清楚地連結到費用頁

#### 12. Tab 在手機的點擊區域偏小
- 課程頁三個 tab（課表時間、舞蹈風格介紹、課程費用）文字很長，手機上擠在一起
- **建議**：縮短 tab 文字（「課表」、「風格介紹」、「費用」）或改用圖示 + 短字

#### 13. 師資頁點擊行為不明確
- Sean 的卡片整張是可點擊的（Link），但視覺上沒有 hover 效果提示
- 新訪客不會知道卡片可以點進去看詳細資訊
- **建議**：加上 hover 效果（overlay 或箭頭圖示）

#### 14. 沒有 404 頁面
- 訪問 `/teachers/kathy` 或任何不存在的路由，顯示的是程式錯誤畫面
- **建議**：新增 `src/app/not-found.tsx`

#### 15. 報名流程全靠 LINE，沒有任何說明
- 點「立即報名」跳 LINE，但沒有任何說明（要說什麼？要準備什麼資料？）
- **建議**：報名按鈕旁或 FAQ 加入「報名步驟」說明（點 LINE → 告知想上的課程 → 確認時間）
